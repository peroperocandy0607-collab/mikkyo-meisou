# 密教相承 梵輪開眼瞑想法（bonrin-kaigen）

白背景・シンプル・カード型のデザイン軸（参考: stellar-squirrel-b3416c.netlify.app）に、
密教瞑想の実践コンテンツ（参考: peroperocandy0607-collab.github.io/mikkyou-meisou）を統合した静的サイト。

## 構成

- `index.html` — 全セクション（梵輪Music / 5つの手順 / 視る練習 / 空海と密教 / 好転反応対応表 / 瞑想日記）
- `styles.css` — Tailwind CDNで足りない独自スタイル（チップ、タイムライン、ステップ開閉など）
- `app.js` — 全ロジック（Web Audio再生・日記の保存/振り返り）
- `images/mandala.jpg` / `images/mandala-center.jpg` — 曼荼羅の画像（JPEG圧縮済み、合計約850KB）

ビルド不要（Tailwind CDN + 素のJS）。ライブラリのダウンロード・npm installなどは一切不要です。

## 検索エンジンに載せない設定

`<meta name="robots" content="noindex, nofollow">` を設定済みです。robots.txtでのDisallowはあえて行っていません（クローラーがページ自体を読めなくなり、noindexタグが認識されず外部リンク経由で説明なしのURLだけが検索結果に残ってしまうことがあるため）。この設定だけで検索結果には表示されなくなります。

## 外部依存について

Tailwind CDN（cdn.tailwindcss.com）とGoogle Fonts（fonts.googleapis.com）の2つの外部サービスのみに依存しています。どちらもClaude/Anthropicとは無関係の、世界中の多数のサイトが使う安定した公開サービスです。デプロイ後のこのサイトはClaudeやAIモデルの仕様変更と一切関係なく動作し続けます（ランタイムでAI APIを呼び出す処理は含まれていません）。将来的にさらに依存を減らしたい場合は、フォントを自己ホストし、Tailwindを事前ビルドしたCSSに置き換えることもできます。

## 瞑想musicについて

7チャクラ + 「サードアイ開眼専用」の特別瞑想トラックの計8種類。すべてWeb Audio API
でその場で周波数を生成しており、音声ファイルは使っていません（軽量・著作権フリー）。

## 瞑想日記について

②サイトにあった「77個の日次チェックボックス」と「長い自由記述フォーム」を統合し、
チップ選択（体の感覚・心の状態・見えた光）＋一言メモ＋週間ストリーク＋①を参考にした月間カレンダー
に置き換えています。データはブラウザのlocalStorageに保存されます（サーバー不要）。

- 記録がある日はカレンダーのセルにドットが付き、タップするとその日の内容（チップ・メモ・音源・時間）を下の詳細パネルで確認・削除できます
- 「実践の節目」として7日・21日・49日を初七日／三七日／四十九日という仏教の法要の節目にちなんだ目安の目標として提示し、連続日数に応じて進捗バーが伸びます

## GitHub Pagesへのデプロイ

1. このフォルダの中身（`index.html`など）をリポジトリのルート、または`docs/`フォルダに配置
2. GitHubリポジトリの Settings → Pages で公開元ブランチ・フォルダを指定
3. 数分後に公開されたURLで確認

## ローカル確認

```
python -m http.server 8791
```
起動後 http://localhost:8791 を開く（`file://`で直接開くとブラウザによって一部機能が制限される場合があるため、簡易サーバー経由を推奨）。
